# 基于React和Egg.js的移动端旅游电商系统

## Technology stack

**前端**

React@17.x+umi@3.5.18+antd-mobile@2.3.4+axios@0.21.4+less

**后端**

mysql+redis+egg@2.15.1+egg-sequelize@6.0.0

## Project introduce

Client

```
yarn

yarn start
```

Server

```
cd server
```

使用server/app.sql创建数据库，需要安装redis，相关配置在server/config文件中。

运行：

```
npm install

npm run dev
```
